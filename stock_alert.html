📈 Stock Alert Automation

A serverless stock monitoring system built with GitHub Actions and Python.
It runs automatically on a regular schedule and sends an e-mail alert when a stock price hits a target.

⚙️ How it works

  • Runs on GitHub Actions (weekday schedule: 2pm EST)
  • Fetches live stock data using yfinance
  • Evaluates custom price thresholds securely stored in GitHub Secrets
  • Sends email alerts via Gmail SMTP
  • No server required (fully serverless execution)

🧠 Architecture

  GitHub Actions (cron)
          ↓
  Python Script
          ↓
  yfinance API
          ↓
  Rule Engine (threshold check)
          ↓
  SMTP Email Alert

🔐 Security

  • All secrets stored in GitHub Secrets
  • No hardcoded API keys or credentials
  • Isolated execution per workflow run

🚀 Tech Stack

  • Python 3.11
  • GitHub Actions
  • yfinance
  • SMTP (Gmail App Password)

📊 Status

🟢 Active — running on GitHub Actions (weekdays)
